# 📈 AI & Data Science Salary Analysis (2024-2026)

## 📌 Project Overview
This project analyzes modern trends in the IT job market, specifically focusing on Artificial Intelligence and Data Science roles. As a Junior Data Analyst with a background in **Psychological Diagnostics**, I apply analytical methods to understand professional growth and compensation patterns in the tech industry.

## 🛠️ Tech Stack
- **Language:** Python
- **Libraries:** Pandas (Data Cleaning), Seaborn & Matplotlib (Visualization)
- **Environment:** Visual Studio Code / Jupyter Notebooks

## 🔍 Key Insights
1. **The Leadership Premium:** Research Team Leads and Analytics Managers are the highest-paid roles, often exceeding $400k annually.
2. **Career Growth:** There is a significant salary jump (approx. 40-60%) when moving from Entry-level (Junior) to Mid-level positions.
3. **Market Focus:** The data suggests that "Engineering" titles currently outpace "Analyst" titles in terms of base compensation.

## 📁 How to run
1. Clone this repository.
2. Install dependencies: `pip install pandas seaborn matplotlib`
3. Open `projekt.ipynb` in VS Code and run all cells.
